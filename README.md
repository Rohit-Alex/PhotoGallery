Gallery App:
The Layout goes like:

> src has App.js
> Components folder has:
>
> > Header(header part)
> > Images(all the images)>>>Image(single image)
> > Popup(for displaying image when clicked)
> > Loading(for loading process)

Features:

> Sticky header, whoes background color changes on scroll event
> Image container displaying images like pinterest feed i.e every row has 5 elements
> On Clicking the image, it shows the full image with option of sliding, to get to previous or next image.
> Image enlarges on hovering over it.

]
